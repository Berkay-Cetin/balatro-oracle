A mystical fortune-telling web app built around the world of [Balatro](https://www.playbalatro.com). Draw a card, reveal your fate — powered by AI and the actual shader from the game itself.
 
---
 
## ✨ Features
 
- **Balatro's Original Shader** — The fluid, swirling background is a direct WebGL port of the GLSL shader used in Balatro, originally authored by [localthunk](https://www.playbalatro.com). Ported from [Shadertoy](https://www.shadertoy.com/view/XXtBRr) to a React/WebGL canvas component.
- **150+ Joker Cards** — Every joker from the base game with full effect descriptions and rarity tiers.
- **AI Fortune Reading** — Draw a card and receive a mystical, poetic fortune reading in Turkish, generated by **Llama 3.3 70B** via the Groq API.
- **Animated Card Stream** — Cards continuously scroll across the screen. Hover to pause, drag upward to select.
- **Full Collection Browser** — Browse all Jokers, Tarot, Planet, and Spectral cards with search and rarity filters.
- **Glassmorphism UI** — Dark, neon-accented interface inspired by Balatro's aesthetic.

## 🚀 Getting Started
 
### 1. Set up environment variables
 
Create a `.env` file in the project root:
 
```env
VITE_GROQ_API_KEY=your_groq_api_key_here
```
 
Get your free Groq API key at → https://console.groq.com
 
## 🎮 How to Use
 
1. **Home Page** — Watch the cards stream across the screen
2. **Hover** over a card to pause the stream
3. **Click** or **drag upward** to select a card
4. Hit **"Falı Aç"** to reveal your AI-generated fortune
5. Visit **Koleksiyon** to browse all cards in the game

## 🖼 The Shader
 
The animated background is a WebGL implementation of Balatro's signature fluid shader:
 
- **Original author:** localthunk ([@localthunk](https://twitter.com/localthunk)), creator of Balatro
- **Shadertoy port:** https://www.shadertoy.com/view/XXtBRr
- **Implementation:** Ported to a React `<canvas>` component using raw WebGL (`BalatroBackground.tsx`)
 
The shader uses layered sine-wave distortions with a spin effect to create the characteristic liquid metal look of Balatro's menus.
